# 闪练AI统一技能训练系统架构

## 概述
本系统采用现代化的架构设计，实现了原子技能、技能库和统一练习管理三大核心组件的有机结合，支持预设技能和用户自定义技能，具备强大的扩展性和灵活性。系统集成了AI提示词优化和原子评分系统，能够根据用户表现生成个性化训练内容。通过精准的原子技能评估和智能化的提示词生成，系统能够持续优化用户体验并提供针对性的训练建议。

## 系统架构

### 1. 原子技能管理系统 (Atomic Skill Manager)
- **职责**：管理最小单位的技能要素
- **功能**：
  - 原子技能的增删改查
  - 相似度匹配算法，自动检测重复技能
  - 技能分类管理
  - 用户原子技能得分追踪
  - 原子技能与高层级技能的关联管理
- **核心特性**：
  - 基于文本相似度的自动去重机制
  - 支持多种应用场景的分类体系
  - 实时更新用户原子技能表现
  - 基于原子技能得分的个性化训练生成
  - 原子技能评分数据的持久化存储

### 2. 技能库管理系统 (Skill Library Manager)
- **职责**：管理用户可练习的技能类型
- **功能**：
  - 预设技能管理（演讲、面试、编程等）
  - 用户自定义技能创建
  - 动态提示词模板管理
  - 与原子技能的关联管理
  - AI提示词优化和个性化推荐
- **核心特性**：
  - 智能原子技能匹配
  - 统一的提示词模板系统
  - 灵活的参数配置
  - 针对性AI提示词生成
  - 支持自定义评估标准

### 3. 通用AI评估器 (Universal AI Evaluator)
- **职责**：根据技能类型和提示词进行内容生成和评估
- **功能**：
  - 通用内容生成功能
  - 标准化的评估结果解析
  - 原子技能得分提取
  - 错误处理和降级机制
  - AI提示词安全格式化
- **核心特性**：
  - 高度可配置的提示词系统
  - 统一的评估结果格式
  - 原子技能评分精确提取
  - 强大的容错能力
  - AI提示词安全格式化
  - 支持多模型适配

### 4. 统一练习管理器 (Unified Practice Manager)
- **职责**：管理所有练习活动的生命周期
- **功能**：
  - 练习会话管理
  - 练习记录持久化
  - 进步趋势分析
  - 用户历史查询
  - 个性化训练内容生成
  - 与标准练习记录格式同步
- **核心特性**：
  - 支持多种技能类型的统一管理
  - 详细的练习历史追踪
  - 智能进度分析
  - 基于原子技能表现的个性化推荐
  - 标准化练习记录格式支持

## AI提示词优化

### 优化策略
1. **安全性优化**：使用安全的字符串格式化方法，避免占位符错误
2. **个性化优化**：根据用户原子技能表现定制提示词内容
3. **完整性优化**：确保提示词包含完整的上下文信息
4. **参数化优化**：动态注入必要的参数和变量

### 实现机制
- 在`universal_ai_evaluator.py`中实现了安全的占位符替换机制
- 支持动态参数注入，包括`target_skill_name`等关键参数
- 使用正则表达式确保只替换存在的占位符

## 原子评分系统

### 评分机制
- **原子技能识别**：系统自动识别练习内容涉及的原子技能
- **多维度评估**：对每个原子技能进行独立评分
- **权重分配**：根据不同技能类型分配相应权重
- **历史追踪**：记录用户在各原子技能上的进步轨迹

### 评分流程
1. AI评估器分析提交内容
2. 识别涉及的原子技能
3. 对每个原子技能进行评分
4. 生成综合评估报告
5. 更新用户原子技能档案

## 数据流

### 技能创建流程
1. 用户定义新技能的需求
2. 系统自动匹配相似原子技能
3. 生成或复用原子技能组合
4. 创建带提示词模板的技能定义
5. 存储到技能库

### 练习执行流程
1. 用户选择技能并启动练习
2. 系统根据用户原子技能表现生成个性化题目
3. 用户提交答案
4. 系统调用AI评估答案并提取原子技能得分
5. 更新用户技能档案和原子技能得分
6. 记录练习历史到标准格式

## API接口

### 练习管理接口
- `POST /api/v1/practice/start` - 开始练习会话
- `POST /api/v1/practice/submit` - 提交答案
- `GET /api/v1/practice/history/{user_id}` - 获取练习历史
- `GET /api/v1/practice/progress/{user_id}/{skill_id}` - 获取技能进步情况

### 技能管理接口
- `GET /api/v1/practice/skills` - 获取可用技能
- `POST /api/v1/practice/skills/custom` - 添加自定义技能
- `POST /api/v1/practice/generate` - 为技能生成内容
- `POST /api/v1/practice/evaluate` - 评估技能内容

### 原子技能接口
- `GET /api/v1/atomic/skills` - 获取所有原子技能
- `GET /api/v1/atomic/users/{user_id}/skills` - 获取用户原子技能得分
- `POST /api/v1/atomic/users/{user_id}/skills/update` - 更新用户原子技能得分
- `GET /api/v1/atomic/users/{user_id}/skills/recommendations` - 获取个性化训练建议

## 扩展性

### 添加新技能类型
1. 定义相关的原子技能
2. 编写相应的提示词模板
3. 系统自动集成到统一框架

### 自定义评估标准
1. 通过提示词模板自定义评估维度
2. 系统自动解析标准化输出
3. 保持与现有系统的一致性

### AI模型扩展
1. 支持多种AI模型（通义千问、豆包等）
2. 可插拔的AI客户端设计
3. 统一的AI交互接口

## 优势

1. **模块化设计** - 各组件职责明确，易于维护和扩展
2. **智能化匹配** - 自动检测相似技能，避免重复建设
3. **统一接口** - 不同技能类型共享相同的操作接口
4. **灵活配置** - 通过提示词模板快速适配新场景
5. **数据驱动** - 基于练习历史的个性化推荐和难度调整
6. **精准评估** - 原子技能评分系统提供细粒度反馈
7. **智能优化** - AI提示词持续优化提升用户体验
